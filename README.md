# 💰 Expense Tracker Mobile Application

A secure and user-friendly **Expense Tracker Android Application** developed using **Flutter** and **Firebase**. This project demonstrates modern mobile application development practices, including cloud database integration, user authentication, and real-time data management.

---

## 📌 Project Overview

The Expense Tracker application helps users manage their daily expenses efficiently. Each user has a private account where expenses are securely stored using Firebase Authentication and Cloud Firestore.

The application was developed as a portfolio project to demonstrate practical skills in Flutter, Firebase, mobile UI development, and cloud-based data management.

---

## ✨ Features

* 🔐 Secure User Authentication

  * User Registration
  * Login
  * Logout
  * Password Reset

* 💵 Expense Management

  * Add Expense
  * Edit Expense
  * Delete Expense

* 📊 Dashboard

  * View Total Spending
  * Monthly Spending Summary
  * Expenses Grouped by Date

* 🔍 Search

  * Search expenses by title
  * Search expenses by category

* 📈 Analytics

  * Category-wise expense summary

* ☁️ Cloud Database

  * Firebase Authentication
  * Cloud Firestore
  * Real-time data synchronization

## 🚀 How to Run

1. Clone this repository.
2. Install Flutter dependencies.

```bash
flutter pub get
```

3. Create your own Firebase project.
4. Add your own **google-services.json** file inside:

```text
android/app/
```

5. Run the application.

```bash
flutter run
```
